# Mod de traduction du jeu Beholder 2

## Pourquoi ce mod ?

Contrairement au premier et au troisième opus, Beholder 2 n'a pas été publié en français, ce qui peut rebuter pas mal de joueurs qui ne parlent pas l'anglais, ou qui simplement veulent jouer pour se détendre et n'ont pas envie de "jouer" avec un traducteur à côté ou passer à côté d'éléments importants du jeu.

Constatant qu'il avait existé un mod de traduction du jeu en français mais que celui-ci n'est plus disponible, voici donc mon mod qui traduit le jeu en français pour permettre à ceux qui le souhaitent de profiter du jeu dans la langue de Molière.

_Le mod n'est pas parfait, mais je suis ouvert aux suggestions d'ajout et/ou de correction._

## Installation du mod

L'installation du mod est très simple et se fait en quelques clics. Il vous suffit de télécharger le dossier `fr`.

Rendez-vous dans le dossier du jeu. Pour trouver son emplacement, voici comment faire :

> **Steam** : Dans votre bibliothèque, clic-droit sur le jeu -> _Gérer -> Parcourir les fichiers locaux_

> **GOG** : Dans votre bibliothèque, sélectionner le jeu -> _clic sur les paramètres tout en haut (à côté du bouton jouer) -> Gérer l'installation -> Montrer dossier_

Une fois dans le dossier du jeu, collez le dossier `fr` dans : ```Beholder2_Data\StreamingAssets\Localization```.
